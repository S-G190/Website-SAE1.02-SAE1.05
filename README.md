Ce projet scolaire consiste à créer un site web interactif présentant une base de données et son utilisation à travers un Système de Gestion de Base de Données (SGBD). L’objectif est de montrer comment les données sont structurées, manipulées et affichées via des requêtes SQL, tout en offrant une interface claire pour explorer et comprendre le fonctionnement d’un SGBD dans un contexte concret.
